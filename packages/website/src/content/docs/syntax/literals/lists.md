---
title: Lists
sidebar: { order: 3 }
---

Lists are expressions surrounded by square brackets and separated by commas. They represent collections of things.

```rip
[1, 2, 3]
```

Lists allow any type of expression to be written in literal syntax. The type of a list is written as `List<T>`, where `T` is a union of the types of all members in the list.
